# Day 2: Installing Linux for Complete Beginners

## Who This Guide Is For

This guide is for regular people who may not feel comfortable with computers yet.

It is for the parent, the worker changing careers, the person who feels behind with technology, or anyone who wants to understand computers better.

You do not need to be technical to start.

The goal today is simple:

Learn what Linux is, why it matters, and how to install it in a safe practice environment.

## Why We Are Learning Linux

Linux is an operating system.

An operating system is the main software that helps a computer run.

Windows is an operating system.
macOS is an operating system.
Android is an operating system.
Linux is also an operating system.

Linux matters because it is used in many places behind the scenes, including websites, servers, cybersecurity tools, smart devices, and home lab environments.

Learning Linux helps you understand how computers work instead of only clicking buttons without knowing what is happening.

## Example

Think about learning basic car maintenance.

You do not need to become a mechanic to understand how to check tire pressure, add washer fluid, or recognize when something sounds wrong.

Linux is similar.

You do not need to become a computer expert.

You are learning basic skills so computers feel less mysterious.
