## Who This Is For

This guide is for people who may not consider themselves “tech people.”

You do not need to know how computers work.  
You do not need to know cybersecurity.  
You do not need to memorize anything today.

The goal is simple:

Learn what Linux is, why it matters, and how to use a few basic commands without fear.

## What Is Linux?

Linux is an operating system.

An operating system is the main software that helps you use a computer.

Windows is an operating system.
macOS is an operating system.
Android is an operating system.
Linux is also an operating system.

Everyday analogy:

If your computer is like a house, the operating system is like the person who manages the house. It helps open doors, turn on lights, organize rooms, and make sure everything works together.
