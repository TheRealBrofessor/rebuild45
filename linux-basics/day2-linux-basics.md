## What Is Linux?

Linux is an operating system.

An operating system is the main software that helps a computer run.

You may already know some operating systems:

- Windows runs on many home computers
- macOS runs on Apple computers
- Android runs on many phones
- iOS runs on iPhones
- Linux is another operating system

Linux may look different at first, but it is not magic.

It is just another way to use a computer.

---

## Is Linux Better Than Windows or Mac?

Linux is not automatically better than Windows or Mac.

They are different tools for different jobs.

Windows is common in homes, schools, and offices.

macOS is common for people who use Apple computers.

Linux is common in servers, cybersecurity tools, cloud systems, and home lab environments.

For everyday computer use, Windows or macOS may feel easier.

For learning cybersecurity, networking, servers, and home labs, Linux is often the better learning tool.

That is why we are using it in REBUILD45.

---

## Simple Comparison

| System | Good For | Why People Use It |
|---|---|---|
| Windows | Everyday home and office use | It is familiar and widely used |
| macOS | Apple users and creative work | It is polished and works well with Apple devices |
| Linux | Learning, home labs, servers, and cybersecurity basics | It gives more room to practice and understand how computers work |

## What We Are Doing Today

Today we are going to try Linux from a USB drive.

A USB drive is a small device that plugs into your computer.

Most people use USB drives to move photos, documents, or work files from one computer to another.

For this lesson, we are going to use a USB drive in a different way.

We are going to place Ubuntu Linux on the USB drive so the computer can start from it.

This lets us look around Linux without replacing Windows or macOS.

When we are finished, we can shut the computer down, remove the USB drive, and start the computer normally again.

Ubuntu’s official documentation says Ubuntu Desktop can run from a USB drive so you can try it without making permanent changes to your computer.

---

## Example

Think of it like test-driving a car.

You have not bought the car.

You have not traded in your old car.

You are just trying it first.

Trying Linux from a USB drive works the same way.

You get to explore Linux before making any permanent changes.

## Important Safety Note

Today, we are not replacing Windows.

Today, we are not deleting the computer.

Today, we are not installing Linux permanently.

Today, we are only trying Linux from a USB drive.

## Step 1: Choose the Right USB Drive

Before downloading anything, choose the USB drive you are going to use.

The USB drive should be:

```text
8 GB or larger
Empty or safe to erase
Not used for important files
```

This is important because the USB drive will be erased when we prepare it for Linux.

Do not use a USB drive that has:

```text
Family photos
Work files
School papers
Tax documents
Passwords
Anything you cannot replace
```

Use a blank USB drive if possible.

## Example

Think of the USB drive like a notebook.

If the notebook already has important notes in it, do not erase it.

Use an empty notebook instead.

For this lesson, the USB drive is going to become a Linux practice drive.

---

## Step 2: Go to the Official Ubuntu Website

Ubuntu should be downloaded from the official Ubuntu website.

Do not download Ubuntu from random websites, ads, pop-ups, or file-sharing pages.

Use the official Ubuntu website:

```text
https://ubuntu.com/download/desktop
```

This page is where Ubuntu Desktop can be downloaded.

Ubuntu Desktop is the version we want because it gives us a normal-looking screen with windows, icons, settings, and apps.

That makes it easier for beginners to explore.

---

## Step 3: Download Ubuntu Desktop

On the Ubuntu download page, look for:

```text
Ubuntu Desktop
```

Then click the download button.

The file you download should end with:

```text
.iso
```

It may look something like:

```text
ubuntu-24.04.3-desktop-amd64.iso
```

The numbers may be different when you download it.

That is okay.

The important things are:

```text
It comes from ubuntu.com
It says Ubuntu Desktop
It ends in .iso
```

---

## Step 4: Understand What an ISO File Is

An ISO file is the file used to install or try Linux.

It works like a digital installation disc.

## Example

Years ago, software often came on CDs or DVDs.

Today, many installers come as downloadable files.

The Ubuntu ISO is the file we will put onto the USB drive.

---

## Step 5: Go to the Official balenaEtcher Website

You cannot simply drag the Ubuntu ISO file onto the USB drive like a normal document.

A special tool has to write the file to the USB drive the correct way.

For this guide, we are using balenaEtcher.

Go to the official balenaEtcher website:

```text
https://etcher.balena.io/
```

Do not download it from random software websites.

---

## Step 6: Download balenaEtcher

On the balenaEtcher website, download the version for your computer:

```text
Windows if you are using Windows
macOS if you are using a Mac
Linux if you are already using Linux
```

balenaEtcher is the tool that will put Ubuntu onto the USB drive the correct way.

---

## Step 7: Understand Why We Need balenaEtcher

You cannot just copy the Ubuntu file onto the USB drive like a photo or document.

That will not make the USB drive start Linux.

balenaEtcher writes Ubuntu to the USB in a special way so the computer can start from it.

## Example

Think of it like making a key.

Having the metal is not enough.

The key has to be shaped the right way before it can open the door.

The Ubuntu file is the metal.

balenaEtcher shapes the USB drive so the computer knows how to start Ubuntu.

---

## Step 8: Confirm the USB Drive Is Safe To Erase

Before opening balenaEtcher, check the USB drive one more time.

Ask yourself:

```text
Is this the correct USB drive?
Is it empty?
Is there anything important on it?
Am I okay with this USB drive being erased?
```

If the answer is no, stop here.

Use a different USB drive.

---

## Step 9: Plug In the USB Drive

Plug the USB drive into your computer.

Wait a few seconds for the computer to recognize it.

If a window opens showing the USB drive, that is normal.

You do not need to copy anything into it.

balenaEtcher will handle the setup.

---

## Step 10: Open balenaEtcher

Open balenaEtcher.

You should see three basic choices:

```text
Flash from file
Select target
Flash
```

This means:

```text
Choose the Ubuntu file
Choose the USB drive
Write Ubuntu to the USB drive
```

---

## Step 11: Choose the Ubuntu File

Click:

```text
Flash from file
```

Find the Ubuntu ISO file you downloaded.

It is probably in your Downloads folder.

Select it.

---

## Step 12: Choose the USB Drive

Click:

```text
Select target
```

Choose your USB drive.

Be careful here.

Make sure you are selecting the USB drive, not another storage device.

If you are unsure, stop and ask for help before clicking anything else.

## Example

This is like choosing which notebook to erase and reuse.

You want to choose the blank notebook, not the notebook with your important notes in it.

---

## Step 13: Click Flash

Click:

```text
Flash
```

balenaEtcher will begin writing Ubuntu to the USB drive.

This may take several minutes.

The computer may ask for permission.

That is normal.

Let the process finish.

When it is done, the USB drive is ready.

---

## Step 14: Restart the Computer

Leave the USB drive plugged in.

Restart the computer.

As the computer starts, you need to open the boot menu.

The boot menu lets you choose what the computer starts from.

Normally, the computer starts from its internal drive.

Today, we want it to start from the USB drive.

---

## Step 15: Open the Boot Menu

Different computers use different keys.

Common boot menu keys include:

```text
F12
Esc
F2
F10
```

On many Windows computers, F12 is common.

On some computers, Esc, F2, or F10 may be used.

On many Macs, you hold:

```text
Option
```

or:

```text
Alt
```

during startup.

You may need to restart and try again if you miss the timing.

That is normal.

## Example

Think of the boot menu like choosing which door to walk through.

One door starts your computer normally.

The other door starts from the USB drive.

Today, we are choosing the USB door.

---

## Step 16: Choose the USB Drive

When the boot menu appears, look for the USB drive.

It may say something like:

```text
USB
UEFI USB
SanDisk
Kingston
EFI Boot
Ubuntu
```

Choose the USB option.

The computer should begin starting Ubuntu from the USB drive.

If the computer starts normally instead, do not panic.

Restart and try opening the boot menu again.

---

## Step 17: Choose “Try Ubuntu”

When the Ubuntu screen appears, you may see options like:

```text
Try Ubuntu
Install Ubuntu
```

Choose:

```text
Try Ubuntu
```

This is the safe option for today.

This lets you explore Ubuntu without installing it permanently.

Do not choose Install Ubuntu today.

---
