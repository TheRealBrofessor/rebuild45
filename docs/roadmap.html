REBUILD//45 Learning Roadmap

This roadmap outlines the journey from digital anxiety to digital confidence. Each phase builds on the previous one, moving from foundational concepts to practical skills.

A Word Before We Start

This roadmap is a guide, not a rigid requirement. Everyone learns at their own pace. If something takes longer, that's okay. If you want to move faster, that's okay too. The goal is understanding, not speed.

Phase 1: Digital Confidence

Foundation Building

Build comfort with computers and basic technology concepts.

  • Understanding computers and operating systems
  • What is Linux and why it matters
  • Creating and using bootable USB drives
  • Files, folders, and directory structure
  • Introduction to the terminal and basic commands

Current Status: In Progress (Days 1-2)

Phase 2: Networking Basics

Understanding Connection

Learn how devices connect and communicate on networks.

  • Home networks and what routers actually do
  • IP addresses in plain language
  • Private vs. public IP addresses
  • Wi-Fi security and protocols
  • DNS and how the internet finds websites
  • Basic network troubleshooting

Current Status: In Progress (Day 3)

Phase 3: Cybersecurity Foundations

Protecting Yourself

Learn practical cybersecurity skills that protect your digital life.

  • Password security and password managers
  • Recognizing phishing and social engineering
  • OSINT basics (Open Source Intelligence)
  • Safe browsing habits
  • Two-factor authentication
  • Introduction to ethical hacking fundamentals

Understanding Tomorrow

  • What AI really is (beyond the hype)

How to Use This Roadmap

  1. Start with Phase 1: Begin with Day 1 and work through the material at your own pace
  2. Don't Rush: Spend as much time as you need on each section. Mastery takes time.
  3. Practice: Each lesson includes hands-on exercises. Do them. Don't just read.
  4. Ask Questions: Confused about something? Look it up. Ask in communities. There's no such thing as a dumb question.
  5. Review: Come back and re-read sections. Understanding grows with repetition.
  6. Share: If you find this helpful, share it with others. Learning together makes it better.

Learning Principles

REBUILD//45 is built on these core principles:

1. Familiarity Over Mastery

You don't need to be an expert. You just need to be comfortable. Understanding basics is more valuable than memorizing advanced concepts you'll never use.

2. Plain Language Always

We avoid jargon when possible. When technical terms are necessary, we explain them in plain English. You should never feel talked down to or locked out.

3. Hands-On Learning

Reading alone doesn't build skills. Each lesson includes exercises where you actually do the things we're teaching. Get your hands dirty.

4. Safe Learning Environment

We never ask you to do anything dangerous. We teach you how to explore technology safely, without breaking anything or putting yourself at risk.

5. Your Pace, Your Journey

There's no deadline. If it takes you a month to get through Phase 1, that's perfect. If it takes six months, that's perfect too. This is your journey.
