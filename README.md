# REBUILD//45

A beginner-friendly cybersecurity and digital literacy learning lab focused on helping adults over 40 learn Linux, AI, networking, and cybersecurity fundamentals without intimidation or unnecessary jargon.

## Mission
REBUILD//45 exists to prove that it is never too late to understand modern technology.

This project is designed for adults who feel left behind by:
- Linux
- AI
- Networking
- Cybersecurity
- Modern digital systems

The goal is not turning people into elite hackers overnight.
The goal is removing fear and replacing it with familiarity.
