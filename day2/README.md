# Day 2 — Linux Basics for Complete Beginners

## Who This Is For
This guide is for people who may not consider themselves tech people.

You do not need to know how computers work.
You do not need to know cybersecurity.
You do not need to memorize everything today.

The goal is simple:
Learn what Linux is, why it matters, and how to use a few commands without fear.

## Topics
- What Linux actually is
- Why Linux powers much of the modern world
- Terminal basics
- Files and directories
- Basic commands
